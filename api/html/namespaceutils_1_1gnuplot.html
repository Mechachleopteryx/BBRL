Remove the noise present on the data by merging too close points (if the preceeding point is at a distance of at most 'epsilon' with respect to the first coordinate).

The point replacing a set of similar points is depending of the type of filter used (chosen by 'nft').

Parameters
dataVThe data to filter (must be SORTED by their first coordinate).
epsilonThe filtering threshold value.
nftThe type of filter to apply.
Returns
The filtered data.
