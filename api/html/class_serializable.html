template<typename T >
static void Serializable::checkIn ( Serializable *(*)(std::istream &)  f)
inlinestatic

Register class T in order to allow the usage of 'createInstance(string, istream)' method.

class T must fullfil those conditions:

  • class T must admit a 'T(std::istream)' public constructor.
  • function 'static string T::toString()' must exist.

Example usage: 'checkIn<T>(&Serializable::createInstance<T>);'
