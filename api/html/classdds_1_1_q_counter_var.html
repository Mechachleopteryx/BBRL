Generate a QCounterVar by EDA, where each individual is evaluated with respect to its performances on the prior distribution when used alone and its distance to variables in 'varList'.

In brief, we want to find a good variable which is also as different as possible than those we already have.

Parameters
nXThe number of states.
nUThe number of actions.
priorThe prior distribution.
NThe number of MDPs to draw when evaluating the individuals.
gammaThe discount factory to use when evaluating the individuals on MDPs.
TThe length of the trajectories when evaluating the individuals on MDPs.
muThe initial mean of the gaussians.
sigmaThe initial st. deviation of the gaussians.
bThe number of individuals to select among the population.
npThe size of the population at each step.
accuracyThe maximal value of the st. deviation of a gaussian before stopping the search.
alphaThe weight of the distance score.
betaThe weight of the update score (of the distance score).
varListA list of variables.
