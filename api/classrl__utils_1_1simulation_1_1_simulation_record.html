Detailed Description

template<typename XType, typename UType, typename RType>
class rl_utils::simulation::SimulationRecord< XType, UType, RType >

Represent the data gathered during the interaction between an Agent and a MDP over a single trajectory.

XType: The type representing a state. UType: The type representing an action. RType: The type representing a reward.
