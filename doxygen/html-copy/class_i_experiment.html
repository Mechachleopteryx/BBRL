template<typename AgentType, typename MDPType, typename SimulationRecordType>
std::vector<double> IExperiment< AgentType, MDPType, SimulationRecordType >::computeDSRList ( double  gamma = -1.0) const

Compute and return the list of discounted sum of rewards of the simulations discounted by 'gamma' (in ]0; 1]).

If 'gamma' is not provided (or invalid), the discount factor used during the simulations will be used instead.

'gamma' has to be provided if you have not saved the trajectories.

Returns
The list of discounted sum of rewards of the simulations, discounted by 'gamma' (in ]0; 1]).
